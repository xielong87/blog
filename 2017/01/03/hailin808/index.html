海林808温控器控制模块，包含面板id设定，开机，关机，温度调节，开关机状态反馈，设置温度反馈，当前室温反馈;
Inputs:
数字量TurnOn TurnOff为开关机。
数字量AcUp，AcDown对应调节温度加减，数字量_26Dge输入为一键设置为26度，主机上电AcDge默认在26度基础上进行增减，下次在这次基础上增减。
数字量OscillatorCheck为宏外部过来的Oscillator定时脉冲信号，最小得1.6秒一个周期，建议设置为hi_time 0.1s low_time 2s;
串行量StringIn为面板回码，此处用来显示开关机状态，室温和设置温度。

Outputs:
模拟量输出AcDge为当前调节的温度，可以用于动态显示温度调节，一般和面板真实设置温度偏差不大；
模拟量输出IndoorTemp为当前面板读取到的室温;
模拟量输出SetTemp为读取的当前面板设置温度；
数字量AcStatus为输出当前开关机显示。
串行量StringOut为输出串码给面板；
串行量AcNow输出对应开机、关机和失去联系的ON、 Off、 Err字符串(30秒没有收到开机或关机回码时显示Err)。

String_Function ProcessString(string StringA)
{
String StringB[30];
integer CRC, char, i, j, xflg;
CRC = 0xFFFF;
for(i=1 to len(StringA))
{
char = byte(StringA,i);
for(j=1 to 8)
{
xflg = (CRC ^ char) & 0x0001;
CRC = (CRC >> 1);
if(xflg <> 0)
CRC = CRC ^ 0xA001;
char = char >> 1;
}
}
StringB = StringA + chr(low(CRC)) + chr(high(CRC));
return (StringB);
}

Integer_Function SetAcTemp(integer PanelNum,integer AcDgr)
{
StringTemp = chr(PanelNum) + “\x06\x00\x01\x00” + chr(AcDgr);
StringOut = ProcessString(StringTemp);
Return (0);
}

PUSH AcUp
{

{
AcTemp = AcTemp + 1;
AcDge = AcTemp;
}
SetAcTemp(PanelId,AcDge);
}

PUSH AcDown
{
if(AcTemp>5)
{
AcTemp = AcTemp - 1;
AcDge = AcTemp;
}

}
